FOREIGN LANGUAGE LEARNING RULES

Semua peserta diwajibkan membaca dan mematuhi peraturan program belajar bahasa asing.

1. GUNAKAN IDENTITAS YANG SOPAN

Gunakan nama yang sopan dan tidak menyinggung peserta lain.

2. SALING MENGHORMATI

Hormati semua peserta dari negara atau bahasa yang berbeda.

3. FOKUS BELAJAR

Gunakan room untuk belajar bahasa, latihan berbicara, dan diskusi.

4. DILARANG TOXIC

Dilarang menghina, mengejek aksen, atau merendahkan kemampuan bahasa peserta lain.

5. PRIVASI & KEAMANAN

Dilarang menyebarkan data pribadi peserta tanpa izin.

6. IKUTI ARAHAN HOST / ADMIN

Peserta wajib mengikuti arahan host, admin, atau mentor saat sesi belajar berlangsung.

SANKSI / PENALTIES

• Teguran / Warning
• Mute dari Room / Room Mute
• Dikeluarkan dari kelas / Removed from Class
• Blacklist Program / Program Blacklist
